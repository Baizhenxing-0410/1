

# IP协议课后练习题：





| 1、IP报头中的TTL字段是用来做什么用的？                       |
| ------------------------------------------------------------ |
| TTL（time to live），即生存时间，该字段表示一个IP数据报能够经过的最大的路由器跳数，TTL字段是由IP数据报发送端初始设置的，每个处理该数据报的路由器都需要将其 TTL值减1，即当一个IP数据报每经过一个路由时，其TTL值会减1，当路由器收到一个TTL值为0的数据报时，路由器会将其丢弃 |
